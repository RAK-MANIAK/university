Мова тегів

HTML - це мова тегів, засобами якої здійснюється розмічання веб-сторінок для мережі Інтернет. Браузери отримують HTML-документи з веб-сервера 
    або з локальної пам'яті й передають документи в мультимедійні веб-сторінки. HTML описує структуру веб-сторінки семантично і початково підказки для відображення документа.
Для того щоб створити таку сторінку, введіть наступний код:
<!DOCTYPE html>
        <html>
        <head>
        <meta charset="utf-8">
        <title>HTML</title>
        </head>
        <body>
        <h2>Мова тегів</h2>
